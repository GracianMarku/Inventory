# Inventory Management System

This is my first full-stack project built while learning ASP.NET Core and backend architecture concepts.

The goal of this project was to understand:

- Layered Architecture (Controller → Service → Repository)
- Repository Pattern
- DTO usage
- REST API development
- Connecting a .NET backend with a JavaScript frontend

---

## Tech Stack

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server 
- Repository Pattern
- Service Layer
- DTOs
- EF Core

### Frontend
- Vanilla JavaScript
- Vite
- Tailwind CSS

---

##  Features

- Category CRUD
- Product CRUD
- Stock In / Stock Out
- Basic Sales module


##  Learning Purpose

This is a learning project and will continue to improve over time.
